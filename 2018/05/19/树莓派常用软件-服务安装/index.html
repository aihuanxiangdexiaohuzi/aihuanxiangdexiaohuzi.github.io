安装常用软件及服务，让树莓派变得强大起来！！^_^_

FTP服务

树莓派默认没有安装ftp服务的，使用命令安装
sudo apt install ftp
然后使用命令连接测试
ftp 192.168.1.39 （这里我用的局域网ip测试）
ftp: connect: Connection refused 报错了
sftp 192.168.1.39
Connected to 192.168.1.39.成功了
这里猜测使用ftp不成功的原因是ftp需要开启一个随机端口，而sftp不需要。
经过百度我发现了事实并不是这样。(打脸了)

参考：ftp 和sftp区别
然后我在服务器树莓派上安装了vsftpd
sudo apt install vsftpd
连接测试。成功了

如果你也有两台树莓派，他们之间想互相实现ftp传输，都需要安装vsftpd，其实在大多数情况下使用sftp也是够用了，毕竟少安装一个软件对空间占用比较小吧233

安装了ftp服务器的好处就是，以后不用再拔插sd了，如果你使用的一台windows电脑，想传点东西到树莓派上，直接使用filezilla软件，直接拖到树莓派里面，快捷，方便。

mplayer 命令行播放音乐

本人使用树莓派外接一个音响，没有使用图形界面，用的远程操作的。

  1. 安装mplayer音乐播放软件
    sudo apt install mplayer
    回车等待安装成功
  2. 使用
    mplayer xxx.mp3(xxx 为你的歌曲名)
    当然你也可以使用
    mplayer *.mp3
    批量播放以.mp3结尾的音频文件

使用上面的命令时，确定是在音频文件夹里使用的命令，否则得使用绝对路径
使用小快捷
9/0 音量加/音量减
↑/↓ 快进/快退
←/→ 慢进/慢退
Enter 下一首
q 退出
空格/p 暂停

mplayer的强大之处在于除了可以播放音频文件，还可以播放视频文件，操作方法和播放音频文件一样的，但是要看到视频，你需要一块屏幕，或者用vnc远程。更多操作查看help
