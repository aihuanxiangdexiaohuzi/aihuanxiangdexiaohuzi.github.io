本文分享我是如何用vscode来写markdown文章的

我写markdown文章主要用于我的个人博客，是用github-pages搭建的，然后在本地搭建了hexo+git方案。

  1. 打开vscode 使用快捷键 ctrl+shift+` 打开命令行窗口，shell选择使用bash

  2. 切换目录到hexo目录 hexo new "xxx" 创建一个带模板的md文章

  3. 打开刚才创建的文章进行编辑，同屏打开一个预览窗口

    这里需要安装插件，由于我使用的是github-pages风格的文章，安装了下面的几个插件
  4. 编辑好后就是上传到博客了
