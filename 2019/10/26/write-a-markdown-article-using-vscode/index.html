使用vscode来写markdown文章

很久一段时间没有更新我的文章了，我的兴趣点总是飘忽不定，三分钟热度说的就是我了，尽管我知道这样做很危险。多年来我已习惯了这种浅尝辄止的做事风格。

三分热度

我总是想的很多，每当我想尝试一件事情时，激情高涨，埋头就是几个小时，甚至有时晚饭都忘记吃。
但是随着时间的推移，事情就开始变得无聊了起来，再加上内容开始变得高深，有时遇到一个问题，折腾几个小时都解决不了，兴奋劲儿就下来了，我就想着放放，过段时间也许来了灵感就解决了呢？可这一放啊！就又被其他的事情吸引住了，兴趣又跑到其他地方去了，就这样周而复始的，这样学学，那样搞搞，最后一样也不精通，一样也不明白。沦落成思想的高手，行动的矮子。

诱惑

外界总是充斥着形形色色的诱惑，坚持一件事有时真的很难。大学里我在一个科技社团里学习了一些电子和编程相关的东西，出来工作后，我的那些技能大部分都随着时间的打磨，工作，人情上的各种琐事消耗着，每每下班回家的几个小时，我会翻出我那生了锈的技能，捣鼓捣鼓一番，自娱自乐。有时我很想把我使用中遇到的各种问题都分享出来。我尝试坚持去做一些有意义的事。

到目前为止，我仍然还是一个编程小白，因为我的三分钟热度，无法深层的挖掘高级的技能，我尝试着改变这种行动。

这篇文章使用vscode来编写，使用vscode来写markdown文章只是一种尝试，可能以后会更多的使用vscode。vscode支持多种编程语言的编写调试，友好的windows用户的体验。

以后将会多多分享我使用的一些心得！！
