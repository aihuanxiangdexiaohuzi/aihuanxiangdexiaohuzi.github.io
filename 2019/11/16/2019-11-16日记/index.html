知识点

  1. 树莓派官方源对于国内用户来说并不友好，为了使我们更好的体验树莓派带给我们的快乐，需要更换国内的源，经网上查到资料，成功更换/etc/apt/sources.list 为自己想要的源。到这本以为就可以了，尝试更新，发现实际还是只有十多k的速度,随后继续翻阅无数资料，直到发现大佬(1)。原来还有一个文件需要改etc/apt/sources.list.d注释了里面的官方源，再更新，秒速更新完成。

  2. lsb_release -a 显示系统版本
